# PRD
## Description
A REST API for managing personal to-do tasks. Users can register, authenticate via JWT, and perform CRUD operations on their own tasks. Built with Python FastAPI and SQLite.
## Features
- User registration and login (JWT-based authentication)
- Create a new task with title and optional description
- List all tasks for the authenticated user
- Update a task (edit title/description, mark as completed)
- Delete a task
## Success Criteria
- All endpoints return appropriate HTTP status codes (201 for create, 200 for success, 401 for unauthorized, 404 for not found, etc.)
- Only authenticated users can access task endpoints
- Users can only access their own tasks
- Task completion status can be toggled
- Tasks persist in SQLite database
- API is documented via OpenAPI (auto-generated by FastAPI)
